# Standalone HTML Çalıştırma

Node/npm kurulumu olmadan doğrudan tarayıcıda çalıştırmak için:

## Seçenek 1 (Dosyayı aç)
- `standalone/index.html` dosyasını çift tıklayıp açın.

## Seçenek 2 (Basit lokal sunucu)
```bash
cd standalone
python3 -m http.server 8080
```
Tarayıcıdan: `http://127.0.0.1:8080`
